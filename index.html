Connect Google Drive
Everything in your hub — planner, meals, wardrobe, Auggie — syncs to a private file in your Google Drive. Open the same HTML file on any device and your data loads automatically.

To get credentials (~5 min):
1. Go to console.cloud.google.com
2. APIs & Services → Library → enable Google Drive API
3. Credentials → + Create Credentials → API Key → copy it
4. Credentials → + Create Credentials → OAuth Client ID → type: Web application → add http://localhost and your file's URL to Authorized JavaScript Origins
5. Paste both below
